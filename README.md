# Tauri + Tanstack

This project is an attempt to create a reusable starter for a Tauri Native app using Tanstack Router and React as its frontend.

Front-end opinionated inclusions:

- `typescript` - language
- `pnpm` - package manager
- `rsbuild` - build tool
- `rstest` - testing tool
- `biome` - linter/formatter
- `vanilla-extract` - CSS-in-JS styling
- `react` - UI library
- `@tanstack/router` - application routing, file-based approach
- `zustand` - state management
- `react-hook-forms` - form util
- `until-async` - async util, avoid unnecessary try/catch hells
- `use-debounce` - debounce util

Cross-Boundary opinionated inclusions:

- `tauri-plugin-zustand`/`@tauri-store/zustand` - store plugin (unofficial)
