# Tauri + Tanstack

This project is an attempt to create a reusable starter for a Tauri Native app using Tanstack Router and React as its frontend.

On the frontend side, it will have several opinionated inclusions:

- `typescript` - language
- `bun` - package manager (no runtime as Tauri is the backend)
- `@tanstack/router` - application routing, file-based approach
- `rsbuild` - build tool
- `rstest` - testing tool
- `biome` - linter
- `vanilla-extract` - CSS-in-JS styling
